🌱 Step 1: Collect Texts

  1. Open Perseus Scaife
  2. Search for any text (Homer, Plato, etc.)
  3. Copy the text or download XML
  4. Save in your corpus-builder

🔍 Step 2: Parse & Annotate

  1. Use LavidasParser for basic parsing
  2. Apply CLTK or Stanza for morphology
  3. Upload to INCEpTION for annotation
  4. Export in CoNLL-U format

🤖 Step 3: AI Analysis

  1. Apply Ancient Greek BERT for embeddings
  2. Use LightSide for classification
  3. Detect patterns with GlossaContactLab
  4. Visualize with your tools

📊 Step 4: Share Results

  1. Push to GitHub repositories
  2. Update this website
  3. Export visualizations
  4. Celebrate your progress! 🎉
